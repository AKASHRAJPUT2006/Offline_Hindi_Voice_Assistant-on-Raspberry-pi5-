This is offline hindi voice assistant built on Raspberry pi 5.
I have used vosk models for automatic hindi speech recognition.
To understand the intent of the user, I built intent file.
To convert text to speech so that the assistant can be able to speak hindi, I used espeak-ng.
# Steps to build this project. Use the files in the following order...
1) update_upgrade
2) audio_library
3) python_environment
4) vosk_models
5) virtual_environment
6) Test.microphone.py
7) main.py
8) intent.py
9) TTS_setup
10) Run_assistant
